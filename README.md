# 🛒 Household Grocery List

A shared CLI grocery list app for multiple household members, built with Python and SQLite.

## Requirements
- Python 3.8+

## How to Run
Open CMD, locate the folder inside and type **python grocery.py**. You can try clicking the Python file as well.

The database (`grocery.db`) is created automatically on first run.

## Features
- Multiple household members
- Multiple named lists (e.g. "Weekly Shop", "Party")
- Lists are shared — everyone sees and edits the same list
- Add, delete, and mark items as bought
- Read-only view mode

## Usage
1. Run the app
2. Select your name from the main menu (or add yourself under **Manage Members**)
3. Pick a list to open (or create one under **Manage Lists**)
4. Add items, mark them as bought, or delete them
